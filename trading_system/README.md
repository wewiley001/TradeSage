# TradeSage ◈ AI-Powered Paper Trading Analytics

A Windows desktop application that aggregates real-time market data and news,
runs LLM-powered predictive analysis, executes paper trades via Alpaca,
and continuously learns from trade outcomes.

---

## Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. (Optional) Install Ollama for Local LLM
Download from https://ollama.com and run:
```bash
ollama pull llama3.1:8b
```

### 3. Launch the App
```bash
python main.py
```

### 4. Configure API Keys
Go to the **Settings** tab and enter your API keys:

| Key | Where to get it |
|-----|----------------|
| Alpaca API Key + Secret | https://alpaca.markets (free paper trading account) |
| Marketstack | https://marketstack.com (free tier: 100 req/mo) |
| Stockdata.org | https://www.stockdata.org (free tier available) |
| NewsData.io | https://newsdata.io (free tier: 200 req/day) |
| AP News | https://developer.ap.org |

## Architecture

```
main.py                   ← Entry point
config.py                 ← All settings, persisted to config/settings.json
data/
  fetcher.py              ← All API clients (Alpaca, Marketstack, Stockdata,
                            NewsData, AP News, Portfolio Optimizer)
core/
  llm_engine.py           ← Flexible LLM provider (Ollama/Anthropic/OpenAI)
                            Prompt construction + signal parsing
  trading_engine.py       ← Risk rules, order execution, trade logging
  learning.py             ← Analyzes trade outcomes, updates LLM prompts
ui/
  app.py                  ← Full PyQt5 desktop UI
logs/
  trades.json             ← All paper trade records
  performance_notes.txt   ← LLM-generated learning notes
config/
  settings.json           ← Saved user settings
```

## Features

### News Sentiment Dashboard
- Live news feed from NewsData.io, Stockdata.org, and AP News
- Per-ticker sentiment scores generated by the LLM
- Overall market mood indicator (Risk On / Risk Off / Neutral)

### LLM Trade Signals
- Analyzes quotes + news + positions together
- Outputs: BUY / SELL / HOLD per ticker
- Confidence score (0-100), reasoning, catalysts, risk factors
- One-click paper trade execution with confirmation dialog

### Paper Trading Simulation
- Connects to Alpaca Paper Trading (free)
- Real portfolio value, cash, positions, and P&L
- Full trade log with entry prices and status

### Performance & Learning
- Win rate, average P&L, trade count
- After 10+ closed trades, the LLM analyzes outcomes
- Generates updated guidance notes fed into future prompts
- System improves its own signals over time

### Risk Controls (Always Active)
- Kill switch: halts all trading if daily loss exceeds limit
- Max position size enforced (default: 5% of portfolio)
- Min confidence threshold before execution (default: 75%)
- Manual HALT button cancels all orders instantly

## Important Disclaimers

- **This is for paper trading only.** The system is configured to use
  Alpaca's paper trading environment and does not place real trades.
- **Not financial advice.** LLM signals are experimental and should not
  be the sole basis for real investment decisions.
- **Past paper performance does not predict live results.**
- Always validate the system extensively in simulation before
  considering any live integration.
